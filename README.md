#  Healthcare Data Analysis and Prediction System

##  Project Overview
This project focuses on analyzing healthcare data to uncover meaningful patterns and build a predictive model for patient admission severity. The goal is to support better decision-making in healthcare by using data-driven insights.

---

##  Objectives
- Clean and preprocess raw healthcare data  
- Perform exploratory data analysis (EDA) 
- Visualize key patterns and trends  
- Build a machine learning model to predict patient admission severity  

---

##  Tools and Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib & Seaborn  
- Scikit-learn  
- Jupyter Notebook  

---

##  Project Structure

##  Data Processing
- Handled missing values  
- Standardized categorical variables (e.g., gender)  
- Converted date columns into usable formats  
- Created new features such as **Length of Stay (LOS)**  

---

##  Exploratory Data Analysis
The dataset was analyzed using various visualizations, including:
- Distribution of patient demographics  
- Length of hospital stay patterns  
- Relationship between gender and hospital stay  
- Identification of trends affecting patient outcomes  

---

##  Machine Learning Model
A classification model was built to predict patient admission severity:
- Model Used: Logistic Regression  
- Data split into training and testing sets  
- Model performance evaluated using accuracy score  

---

##  Results
- The model was able to identify patterns in patient data and make predictions on admission severity  
- Visualizations provided insights into how different factors influence hospital stays and patient outcomes  

---

##  Key Insights
- Certain patient characteristics influence length of stay  
- Patterns in the data can help predict admission urgency  
- Data-driven approaches can improve healthcare decision-making
